# MPC(MpandaComponents) for Vue3

## Installation
```
npm install @mpanda/mpc -S
```
## Usage
```js
import Vue from 'vue'
import MPC from '@mpanda/mpc' 
Vue.use(MPC)
```
## Browser Support
> Modern browsers except IE
